# 小区楼间距

- 楼间距影响中底层采光，特别是冬天时，南面的楼影子较长，容易遮挡北面的楼，造成直接日照时间短
- 按照国家有关住房楼间距的规定，以冬至日照时间不低于1小时（房子底层窗户）为标准。
- 楼间距是用建筑物室外坪至房屋檐口的高度/tan(a)，其中a是指各地冬至日正午时的太阳高度角
- **楼高: 楼间距** = `1 : 1.2`

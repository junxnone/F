# 柜子
- 一般按投影面积计算，抽屉配件玻璃有些另算
- 柜子板材
  - 颗粒板  $1100/m^2$  左右
  - 欧松板 $1300/m^2$  左右
  - 多层板 $1550/m^2$  左右
  - 实木板/生态板  $1675/m^2$  左右

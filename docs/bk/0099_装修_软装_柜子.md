# 柜子

  - 一般按投影面积计算，抽屉配件玻璃有些另算, 也有系统报价，按实际设计板材配件计算
  - 柜子板材
      - 颗粒板 $1100/m^2$ 左右
      - 欧松板 $1300/m^2$ 左右
      - 多层板 $1550/m^2$ 左右
      - 实木板/生态板 $1675/m^2$ 左右
  - 五金
  - 其他可能单独收费的项目
      - 安装
      - 抽屉
      - 玻璃
      - 圆弧
      - 收边条

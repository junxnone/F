# 电器

  - 软装电器要提前确定的原因:
      - 确定软装柜子的尺寸，如冰箱嵌入，电视嵌入，洗衣机嵌入。
      - 需要匹配风格，如灯具。
  - 需要买的电器
      - 冰箱
      - 电视/投影仪
      - 洗衣机
      - 灯具
      - 智能马桶

## 需要考虑的问题

  - 尺寸/ 嵌入式？
  - 打开方式/插座预留
  - 固定方式/移动方式
  - 颜色

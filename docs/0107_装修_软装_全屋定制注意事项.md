# 全屋定制注意事项
- 选择颜色时要板材小样
- 进场安装时通过五金孔查看板材材质
- 检查封边
- 检查板材和五金型号
- 检查板材厚度（背板）
- 检查标签后面有没有划痕破损
- 开孔处要封边条封边
- 安装完后检查
  - 平整度
  - 缝隙
  - 抽屉开关是否顺畅
  - 灯带
